# 🛒 의류 필요성 기반 스마트 위시리스트 (Smart Wishlist)

소프트웨어개발실습1 과제 - 필요성 점수 기반 의류 우선순위 관리 시스템입니다.

## 1. 프로젝트 소개
* 무분별한 쇼핑을 방지하기 위해 사용자가 옷을 등록할 때 '필요성 점수'를 매겨 관리하는 Java Swing 기반 GUI 프로그램입니다.
* 상단 대시보드를 통해 이번 달 쇼핑 예산 및 추천 아이템을 직관적으로 보여줍니다.

## 2. 주요 기능 및 사용법
* **새 위시리스트 추가:** 하단의 `새 위시리스트 추가` 버튼을 눌러 카테고리, 상품명, 가격, 필요성 점수(1~10), 스타일 태그를 팝업창을 통해 입력 및 등록할 수 있습니다.
* **선택 삭제:** 표에서 원치 않는 아이템 행을 선택한 후 `선택 삭제` 버튼을 누르면 리스트에서 실시간으로 제거됩니다.
* **우선순위 자동 정렬 (핵심 기능):** 표 상단의 `필요성 점수 (1~10)` 헤더를 클릭하면 점수가 높은 순(가장 필요한 옷) 또는 낮은 순으로 실시간 정렬이 가능합니다.

## 3. 실행 방법 (How to Run)
1. 이 레포지토리의 `SmartWishListApp.java` 파일을 다운로드합니다.
2. Java 개발 환경(JDK 11 이상 권장) 또는 IDE에서 해당 파일을 엽니다.
3. 프로그램을 실행(`Run`)하면 GUI 창이 활성화됩니다.
